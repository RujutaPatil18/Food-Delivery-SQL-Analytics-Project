# Food Delivery SQL Analytics Project
# Project Overview

This project analyzes food delivery business data using MySQL. The objective is to generate business insights and KPI reports related to customer behavior, restaurant performance, revenue generation, and delivery operations.

# Tools Used
MySQL
SQL
GitHub

# Dataset
The project contains three tables:

# Customers
customer_id

city
# Restaurants
restaurant_id
restaurant_name
cuisine
# Orders
order_id
customer_id

order_date
order_amount
status
delivery_time
